# Daily Reflection Tree & AI Agent

DT Fellowship — R1 System Entry Assignment

---

## 🔍 Problem Statement

This assignment simulates structured thinking using a **Daily Reflection System**.
The goal is to:

* Convert unstructured daily experiences into structured insights
* Build a deterministic decision tree
* Demonstrate how structured input improves AI output

---

## 🧠 Approach

The solution is divided into two parts:

### Part A — Decision Tree (Deterministic)

A structured logic flow that evaluates a user's day based on:

* Task completion
* Learning
* Health
* Distractions

Each condition leads to a **clear, non-random outcome**.

---

### Part B — AI Agent (Optional)

A Python-based agent that:

* Takes user input
* Applies decision logic
* Outputs structured reflection

---

## 🌳 Decision Tree Logic

The system evaluates the day through sequential checks:

1. Were planned tasks completed?
2. Was something new learned?
3. Was physical and mental health maintained?
4. Were distractions controlled?

### Final Output:

* Productive Day
* Average Day
* Unproductive Day

With **specific action recommendations**

---

## 🤖 AI Agent

The agent is a CLI-based system that simulates structured reflection.

### Features:

* Controlled inputs (yes/no)
* Deterministic outputs
* No hallucination
* Actionable suggestions

---

## 🛡️ Guardrails Against AI Hallucination

The system ensures reliability by:

* Using fixed decision logic (if-else)
* Restricting input types
* Eliminating open-ended AI generation
* Producing deterministic outputs only

---

## ⚙️ How to Run

```bash
python agent.py
```

---

## 📌 Key Insight (Core Learning)

> Structured input produces better output than unstructured thinking.

Same user. Same day.
But with structure → clarity increases dramatically.

---

## 🚀 Conclusion

This assignment demonstrates:

* Structured thinking over raw intelligence
* Deterministic systems over vague reasoning
* Practical application of AI with guardrails
