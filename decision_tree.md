# Daily Reflection Decision Tree

## START

### 1. Task Completion

* Did you complete your planned tasks?

  * YES → Go to Learning
  * NO → Identify reason:

    * Time issue → Improve planning
    * Distraction → Reduce interruptions
    * Motivation → Build discipline

---

### 2. Learning

* Did you learn something new?

  * YES → Document learning
  * NO → Allocate learning time

---

### 3. Health

* Did you maintain health (sleep, food, exercise)?

  * YES → Continue
  * NO → Improve lifestyle

---

### 4. Focus

* Were distractions minimized?

  * YES → Good
  * NO → Identify triggers

---

## FINAL CLASSIFICATION

* If all YES → **Productive Day**
* If mixed → **Average Day**
* If mostly NO → **Unproductive Day**

---

## OUTPUT

* Day Classification
* Key Issues
* Action Plan for Tomorrow
